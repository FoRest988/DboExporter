# Dragon Ball OnLine Exporter

## Building:

Use git bash Command:

# 1:
git clone http://github.com/Dbo-Kt69/DboExporter
Or 
git clone https://github.com/Dbo-Kt69/DboExporter

# 2:
cd DboExporter

# 3:
mkdir build

# 4:
cd build

# windows
cmake .. -A Win32

# other
cmake ..

# Open .Sln
Open your build (.\DboExporter\build\DboExporter.sln)
